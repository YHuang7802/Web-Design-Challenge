Summary: Latitude vs X

This was a project examining weather in relation to the proximity to the equator. The data was pulled from the OpenWeatherMap API where 500 cities were randomly selected.

After the data was compiled into a dataset, Matplotlib was used to plot the weather data relative to the latitudes of the cities. The factors of interest were temperature, cloudiness, wind speed, and humidity. This site contains the data and the plots of the analysis.
